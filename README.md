# NewsNest
NewsNest is a modern news website built with React and Tailwind CSS, providing the latest updates across various categories like General, Entertainment, Business, Health, Science, and Technology. The news data is sourced from the News API, ensuring real-time and accurate information.

## Features
- Multiple Categories: Browse news articles in categories such as General, Entertainment, Business, Health, Science and Technology.
- Detailed View: Click on the "Read more" button to read detailed news articles.
- Responsive Design: Optimized for different screen sizes using Tailwind CSS.
- Real-Time Data: Fetches up-to-date news articles from the News API.
- Seamless Navigation: Utilizes React Router for smooth navigation between different news categories.

# Installation
To get a local copy up and running, follow these simple steps:

- Clone the repository: git clone https://github.com/soniyasoni3331/News-Nest.git
- Navigate to the project directory: cd NewsNest
- Install dependencies: npm install
- Create a .env file in the root directory and add your News API key: REACT_APP_NEWS_API_KEY=your_news_api_key
- Start the development server:npm run dev

## Usage
- Open your browser and navigate to http://localhost:3000.
- Explore various news categories using the navigation bar.
- Click on the "Read more" button to view detailed news articles.

## Dependencies
- React - A JavaScript library for building user interfaces.
- React Router - A collection of navigational components for React applications.
- Tailwind CSS - A utility-first CSS framework for rapid UI development.
- News API - A simple HTTP REST API for searching and retrieving live articles from all over the web.

## API
NewsNest uses the News API to fetch the latest news articles. You will need to sign up for an API key and add it to your .env file as shown in the Installation section.
